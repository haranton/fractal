# Fractal Generator (Go)

Учебный проект по генерации фракталов на языке Go.

Программа строит несколько видов фракталов и сохраняет их в бинарном
графическом формате PNG.


## Реализованные фракталы

- Множество Мандельброта
- Множество Жюлиа

Каждый фрактал сохраняется в отдельный PNG-файл.

## Отладочный код

В проекте используется отладочный код на основе проверок условий
(аналог assert):

 - Проверка корректности размеров изображения

 - Проверка успешного создания файлов

 - Проверка корректного выбора типа фрактала

При нарушении условий программа аварийно завершается с сообщением об ошибке.

## Стиль кодирования

Проект следует Google Code Style для Go, а именно:

   - стандартный формат Go (gofmt)

   - единые правила именования

   - читаемая структура кода

Форматирование выполняется автоматически инструментами Go.
